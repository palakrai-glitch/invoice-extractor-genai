# Automated Invoice Data Extractor

## Project Overview

Extracts structured invoice data from PDF invoices using:

- pdfplumber
- Google Gemini API
- Python

## Features

- PDF Text Extraction
- Data Cleaning
- Structured JSON Output
- Handles Missing Fields

## Installation

pip install -r requirements.txt

## Run

python invoice_extractor.py

## Tested Invoices
